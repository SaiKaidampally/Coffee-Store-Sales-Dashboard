# Coffee Store Sales Analysis Dashboard

This project analyzes coffee store sales data using Python, Pandas, and Matplotlib to generate business insights through visual dashboards.

## Project Objective
To understand sales trends, best performing products, monthly revenue patterns, and category-wise performance.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
